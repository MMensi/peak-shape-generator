gaussian

src/gaussian.js

Calculate a normalized gaussian shape

gaussian(options: object): Float64Array
Parameters
options (object = {})
Name Description
options.FWHM number (default 500) number of points in Full Width at Half Maximum, Standard deviation will be computed as FWHM / 2 / sqrt(2 ln(2))
options.SD number? Standard deviation, if it's defined FWHM parameter will be ignored.
options.factor number (default 3) factor of standard deviation to increase the window size, the vector size is 2 factor SD
Returns
Float64Array: array of Y points

halfGaussian

src/halfGaussian.js

Calculate a half gaussian shape

halfGaussian(options: object): Float64Array
Parameters
options (object = {})
Name Description
options.FWHM number (default 500) number of points in Full Width at Half Maximum, Standard deviation will be computed as FWHM / 2 / sqrt(2 ln(2))
options.SD number? Standard deviation, if it's defined FWHM parameter will be ignored.
options.factor number (default 3) factor of standard deviation to increase the window size, the vector size is 2 factor SD
options.ascending boolean (default false) if it is true the vector will contain the left side of the Gaussian shape
Returns
Float64Array: array of Y points
